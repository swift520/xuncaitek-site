# Chang Chen Xun Cai Robotics Data Website

Static bilingual landing page for 福州长辰迅采软件技术有限公司.

## Overview

This site introduces the company's embodied AI data collection and annotation services:

- body-free real-world Ego data collection
- wearable capture and multimodal synchronization
- industrial, household, commercial and service scene sourcing
- annotation, QA and dataset delivery for VLA training
- delivery formats including RLDS, LeRobot, HDF5, Parquet and JSON

## Local Preview

Open `index.html` directly in a browser, or run a simple static server:

```powershell
python -m http.server 4175
```

Then visit `http://127.0.0.1:4175/`.
